# Facebook Account Creator Telegram Bot

هذا البوت يقوم بأتمتة عملية إنشاء حسابات فيسبوك لتجاوز حظر الـ IP.

## المتطلبات
- Node.js
- توكن تيليجرام (Telegram Bot Token)

## التشغيل
1. قم بضبط توكن التيليجرام في ملف `fb_bot.js` أو كمتغير بيئة `TELEGRAM_TOKEN`.
2. قم بتشغيل الأمر:
   ```bash
   npm start
   ```

## المزايا
- توليد أسماء وتواريخ ميلاد عشوائية.
- طلب البريد الإلكتروني أو رقم الهاتف من المستخدم.
- التعامل مع صفحة التسجيل في فيسبوك.
- طلب رمز التحقق من المستخدم وإدخاله تلقائياً.
- تأكيد نجاح العملية وإرسال البيانات النهائية.
